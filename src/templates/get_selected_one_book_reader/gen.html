This is the generator template for BOOK READER. This is used once everytime a new component is created. After that the component is independent of this template. Because the user can edit and modify the component as they wish. This is just a starting point. So, it will not be copied to each component like the TSX template is copied. If this (gen.html) is not found for any component we will use the default one available. Which will be at src/templates/default/gen.html
